Terminal Pro V5.0

掌握雙軌風險驗證、AI 降載接管與雲端對帳的全方位資產管理系統

1. Dashboard / 資產配置

  • ▪ 新增持股：在列表最下方輸入股票代號、目標權重(%)、股數與成本，點擊「加入」。支援台股(如: 2330)、美股指數(如: ^VIX) 與黃金(輸入 AU9901)。
  • ▪ 拖曳排序：滑鼠按住列表左側的 ⋮⋮ 圖示，即可上下拖曳自訂股票的顯示順序。
  • ▪ 編輯與刪除：點擊右側的 ✎ 可修改股數、成本與目標權重；點擊 ✕ 可刪除該檔股票。
  • ▪ 全域 60MA 偵測：系統會自動抓取所有持股的過去 1 年資料，計算出最精準的季線(60日還原權息MA)與乖離率。若乖離小於 5%，會出現黃金買點提示標籤。

2. Macro Monitor / 總經監測

  • ▪ 雙層監控架構：分為「動能與情緒 (賺錢訊號)」與「系統性風險 (保命預警)」兩大層級。系統每 30 秒自動抓取 Yahoo Finance 最新報價。
  • ▪ 殖利率曲線矩陣 (雙重驗證)：整合市場派 (10Y-2Y) 與央行派 (10Y-3M)。當 10Y-2Y 率先轉正時觸發「黃燈警戒」；若 10Y-3M 也跟著轉正，則觸發「紅燈死刑」，宣判海嘯到來。
  • ▪ 統一血條設計：所有指標血條皆設計為「左側偏綠 (買點/安全)」至「右側偏紅 (賣點/危險)」，讓大腦直覺判斷，不需反向思考。

3. Strategy / AI 策略執行

  • ▪ HEAT SCORE (0-100分)：平時由六大技術與籌碼指標運算。分數低代表恐慌（買點），分數高代表貪婪（賣點）。
  • ▪ 宏觀覆寫機制 (Override Protocol)：當「海嘯預警指數」> 30% 時，AI 會自動**降載壓抑**買進訊號；若 > 60% (雙曲線轉正)，AI 將**強制接管**，無視所有動能指標，無條件啟動「海嘯防禦模式」要求清倉避險。
  • ▪ 一鍵下單 (Order Blotter)：系統會根據目前策略，自動換算所有持股「應該買賣多少股」。點擊右側的 → 即可一鍵完成虛擬下單並自動調整現金池。

4. Ledger / 帳務總覽

  • ▪ 資金分流紀錄：分為「入金管理」(本金) 與「配息管理」(收益)。這能讓系統準確計算你真正的投資報酬率 (ROI)。支援列表拖曳重新排序。
  • ▪ 刪除與回補：點擊紀錄右側的 ✕ 刪除該筆流水帳時，系統會自動幫你把「可用現金池」的餘額加回/扣除，確保帳目不亂。
  • ▪ 自動資產對帳：將券商 APP 上的「證券總成本」與「銀行交割戶餘額」輸入對帳中心。若兩者相加「等於」你的總入金+配息，系統會顯示綠色 帳目吻合，防呆防錯。

5. Security & Backup / 系統安全

絕對隱私 (Local Storage)

本系統採無伺服器架構 (Serverless)，您所有的資產數據、股票代號與資金紀錄，都**僅存在您當前設備的瀏覽器中**，任何人(包含開發者)皆無法竊取您的隱私。

中文股名 API 串接

為解決 Yahoo API 無法穩定獲取台股中文名的問題，請至 FinMind 免費註冊並獲取 Token，貼入左側設定中，系統即會自動抓取正確名稱。

強烈建議：定期 GitHub 雲端備份

由於資料存在本地端，若您**清除瀏覽器快取、使用無痕模式或更換手機/電腦**，資料將會消失。

  1. 前往 GitHub 申請 Personal Access Token (PAT)，並確保勾選 `gist` 權限。
  2. 將 Token 填入備份頁面右側，點擊「記憶授權設定」。
  3. 點擊 Upload 一鍵上傳備份。系統會自動產生一組 Gist ID。
  4. 在另一台設備上，填入相同的 Token 與 Gist ID，點擊 Download 即可完美還原所有資料！
